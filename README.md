# 🚀 rewwy-express

Rewwy Express is a self-developed "homegrown" AI chatbox, focusing mainly on core features.

---

## 🌟 Main Features

- **Express 5**: Modern async route handling.
- **TypeScript**: Type safety across the stack.
- **Security**: Helmet, CORS, strong error handling.
- **Rate Limiting**: Efficient API protection.
- **Compression**: Built-in response gzip.
- **Logging**: Morgan, pluggable with Winston.
- **Authentication**: JWT ready for expansion.
- **Redis**: Caching, sessions support.
- **PostgreSQL**: Built with Drizzle ORM, Neon ready.
- **OpenAI**: Simple integration.
- **Config & Build**: .env management, SWC + Webpack for production.
